# 开发移动应用

ThingsPanel提供了开源的App，使用Uniapp开发，可以直接导出成各种小程序以及iOS、Android。
# 移动端当前的功能包括

- 设置登录的IP地址或者服务器。
- 项目和分组筛选设备列表
- 显示设备列表以及在线信息。
- 设备详情，查看当前值与曲线。
- 扫码添加设备到分组。
- 自动化控制。
- 个人账户信息。
  
如果需要更多的功能，需要自行增加。
