# 环境要求

## 硬件设备要求

**最低配置**

|CPU|1核心|
|--|--|
|内存|1GB|
|硬盘|20G|

**推荐配置**

|CPU|2核+|
|--|--|
|内存|4GB+|
|硬盘|40GB+|
