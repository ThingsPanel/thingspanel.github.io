Quickly install and connect devices.

This section guides you to quickly install ThingsPanel and implement access to devices to view device data.

Prequisites​

  • Docker needs to be installed first (to install docker community version)
  • DockerCompose(Install Docker Compose)
  • Git
  • Use the MQTTT Simulator to use MQTTbox, MQTT X, MQTT.fx or use device to push data.
info

More detailedstepsplease refer toDocker Install.

Install ThingsPanel via Docker container​

Deploying with containment is the fastest way to build ThingsPanels.

第一步、获取docker-compose源码:
git clone https://github.com/ThingsPanel/thingspanel-docker.git
第二步、进入目录并启动服务:
cd thingspanel-docker&&docker-compose -f docker-compose.yml up
第三步、登录:
输入网址:服务器IP:8080
系统管理员：super@super.cn / 123456
租户管理员：tenant@tenant.cn / 123456
info

More detailedstepsplease refer toDocker Install.

Steps​

info
  • Add device：to add item, group, input device name.
  • Pushing data：uses a push parameter provided by the system.
  • View data：at the location monitored by the device.

Connect a temperature and humidity device and view data in 30 seconds [suitable for experts]​

Detailed steps for newer​

Create Device​

1, in the Device Admin menu, click the new project button, enter project name, e. g. environmental monitoring.

2, create, and enter device name, select group, set to device(direct device) 3, select device plugins

4, get push parameters click on link view detailed push parameters, use the following parameters to configure the device

Pushing data​

Push Parameter Instructions：

  • Access Token：as the username for MQTT
  • mqtt's service Ip:1883：if the domain name is dev.thingspanel.cn, enter dev.thingspanel.cn, port 1883 (device equivalent) in the Push Model.
  • device/attributes：Topic to push data

Message format.

{
    "temp": "Temperature",
    "humidity": "humidity"
}

Change required

{
    "temp": "28",
    "humidity": "66"
}

Referred to：temperatures of 28 degrees, 66% humidity.temp and humidity cannot be changed!

Configure Connection

After configuration is complete, see the figure below. Other fields can be left blank.： MQTT X connection configuration

When configured, tap to connect to the Mqtt server successfully.

Pushing data

Fill in topics and messages.

View Data​

In the Device Monitoring menu, click to view the newly created environmental monitoring project

Click to show the following graph, you canto view the device data just pushed

You can also view it in the following two menus:

  • Time when the message was last pushed in the Device Admin menu.
  • Detailed data just pushed up in Data management
