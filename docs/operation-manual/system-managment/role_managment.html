角色管理

1、说明

系统管理员可以创建不同的角色(职位)，每个角色(职位)拥有多个不同的权限。

2、操作步骤​

1). 添加角色​

系统管理 -> 角色管理 -> 添加角色
点击添加角色按钮，填入角色(职位)的名称，点击保存。

2). 编辑角色​

系统管理 -> 角色管理 -> 编辑角色
点击编辑按钮，可以修改角色的名称。

3). 权限管理​

系统管理 -> 角色管理 -> 权限管理
点击权限管理按钮，给角色分配权限。

4). 删除​

系统管理 -> 角色管理 -> 删除
系统管理员点击删除按钮，可以删除指定角色。
