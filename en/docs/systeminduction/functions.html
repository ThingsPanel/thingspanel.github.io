Function introduction

Home​

  • Server status information, including CPU and memory usage, number of device messages.

Project​

  • create multiple projects, create project according to the attribute structure, and group them at unlimited levels.

  • A project can bind a user to achieve user authorization for the project.

Device management​

  • Add management devices in groups.Get the device push parameter for data pairs.

  • View the device push status, the chart component of the device plug-in, and quickly filter the device according to the grouping and device plug-in.

  • Provides device wizard for newcomers.

  • Unlimited groupings can be managed.

Visualization​

  • You can create multiple visualizations.

  • A business can create grouped and aggregated visualizations.

  • The chart units contained in the device plug-in are automatically read and loaded.

  • You can set the refresh rate, sample rate, and time interval.

  • Various icon modules, report units are supported.

  • Drag-and-drop layout

Automation​

  • You can add control policies and alarm policies

  • Supports controlling policy priority.

  • Supports multiple control policies or conditions

  • Two trigger types are supported: time and device triggering.

  • Supports multiple control commands to be issued at the same time.

  • Support different policies for different businesses.

Data management​

  • Filter data based on time period based on business.

  • View the received data log in real time to facilitate monitoring whether the data has been received.

  • Data can be exported.

Alarm information​

  • Displays alarm information according to business and grouping.

  • You can filter based on the time period.

System logs​

  • View the access paths of different IPs in the system for security audits.

  • View any records of the device being operated, both manually and automatically, by what conditions triggered it.

Application Management​

  • Manage device plug-ins, which are shadow devices of devices, including data and protocol definitions, data processing, and graphical report units for device output.

Users Management​

  • Casbin scheme supports ACL and RBAC multiple permission management methods, and supports large-scale business permission control.

  • Support permission control to the page.

  • Supports separate permission control for projects.

  • Support multi-role definition and custom role permissions.

Rule Engine​

  • Forward the received data to third parties.

  • Receive device data and convert the format.

Product management​

  • Create products, set product codes, export product production data

  • Import products and generate QR codes

  • OTA functionality.

System Settings​

  • Replace the logo.

  • Change the system title.

  • Change the theme style.

APP​

  • Using Uniapp development, it can be easily compiled into iOS, Android, WeChat Mini Program and other Mini Programs, H5.

  • You can scan the code to add a device (the device needs to be imported in the backend first).

  • View the monitoring values.

  • Switch between intelligent service and device groupings.

  • Manual control.

  • Set the control policy, which is divided into two types: device triggering and time condition triggering.

  • Review the operation log.

  • Personal account management function.

  • Mobile phone verification code login.
