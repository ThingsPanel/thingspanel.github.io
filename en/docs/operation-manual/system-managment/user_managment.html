User management

1. Description​

System administrators can create multiple users (staff), each user (staff) assigned one or more roles (positions), each with different permissions.

2. Operation steps​

1). Add users​

System Administration -> User Management -> Add User
Click the Add User button to fill in the pop-up dialog box with name, email address, phone, 123456 you can use the default password, or you can set your own password.
At the same time, we can assign the corresponding role (position) to the user, and check the corresponding role.
Finally, click Save and the user is created.

2). Edit the user​

System Administration -> User Management -> Edit User
Click the Edit button to modify the user's (staff member's) name, email address, etc., and reassign the user's role (position).

3). Change the password​

System Administration -> User Management -> Modify Password
The system administrator can reset the login password for the user by clicking the Edit button.

4). Delete​

System Administration -> User Management -> Delete
The system administrator can delete the specified user by clicking the Delete button.
