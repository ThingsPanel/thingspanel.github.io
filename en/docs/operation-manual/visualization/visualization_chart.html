Visualize the chart

1. Description​

In the Visualization module of Things Panel, users can customize the visualization chart of the device, as well as the components used to interact with the device.

2. Functionality​

When users enter the visualization chart interface via a visualizable page, they first show the user a general view of the business's equipment.
includes the total number of devices, the number of offline devices, the number of online devices. Figure：

If there is no overall view of the device, you can add a total device chart by clicking the+button in the top right.

Users can display the visualization or interaction they want to view by selecting the Device Group drop-down list

  1. Then click+button to add a chart for that grouping
